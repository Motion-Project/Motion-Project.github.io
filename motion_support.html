Support

If you are having an question or issue with the setup, configuration or use of Motion, the following resources are available to assist.
  • Motion Guide
  • Motion Discussions Page
  • MotionPlus Guide
  • MotionPlus Discussions Page
  • Join the motion-user mailing list and send an email including as much detail as possible with the question. (Use the same link to unsubscribe)
  • #motion on Libera Chat

If you think you have found a bug, please make sure that you are using the latest release version. Note that the version installed via apt may not be the most recent. Check the releases page and ensure the version being used is the most current.

The issues on the github site are to discuss code problems and crashes. Application enhancements are generally only considered for MotionPlus and are documented on the discussions page.
